## 🎯 Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio pessoal. O objetivo principal deste projeto foi construir uma interface web de alto desempenho com um design cinematográfico e premium, refletindo a mesma excelência estrutural que aplico no desenvolvimento de ecossistemas Full Stack.

Ao invés de depender de templates pesados ou frameworks de UI engessados, a arquitetura foi desenhada no modelo **"Zero-UI-Framework"** (Vue 3 puro + CSS Vanilla). Essa decisão garante carregamento instantâneo, total acessibilidade e controle cirúrgico sobre a tipografia, animações e responsividade.

## ✨ Principais Funcionalidades

- **Design Minimalista e Alta UX:** Utilização de microinterações, cursores customizados e um layout baseado em Grid CSS, voltado à máxima legibilidade.
- **Scroll Animado Nativo:** Animações elegantes controladas via `Intersection Observer API` diretamente no ciclo de vida do Vue, evitando o peso extra de bibliotecas como GSAP para tarefas simples.
- **Apresentação de Ecossistema:** Seções divididas detalhadamente para apresentar minhas vertentes de atuação: do Back-end de alto volume (Python, Java), passando pelo Front-end reativo, até a Infraestrutura de servidores e Automação IoT.

## 🛠 Tecnologias Utilizadas

O projeto utiliza as ferramentas mais modernas do ecossistema front-end para garantir segurança e velocidade:

- **[Vue 3](https://vuejs.org/)** — Framework reativo e modular utilizando a nova Composition API.
- **[Vite](https://vitejs.dev/)** — Build tool e bundler ultrarrápido (HMR nativo).
- **CSS Vanilla (CSS3)** — Uso restrito de variáveis para manipulação avançada de cores (Dark/Light themes contextuais) e design system próprio.
- **[GitHub Pages](https://pages.github.com/)** — Hospedagem do bundle estático.

## 🚀 Como executar o projeto localmente

Caso queira clonar a aplicação e rodá-la em sua própria máquina, siga as instruções:

### Pré-requisitos
- **Node.js** (v16.0 ou superior)
- **NPM** ou **Yarn**

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/igorfigueiredobrito/portfolio.git

# 2. Acesse a pasta raiz do projeto
cd portfolio

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento local
npm run dev
```

A aplicação estará rodando imediatamente no endereço `http://localhost:5173`.

## 📦 Build para Produção

Para gerar uma versão minificada e otimizada pronta para o deploy em produção, execute:

```bash
npm run build
```
O Vite irá transpilar os componentes e empacotar todos os arquivos estáticos otimizados na pasta `/dist`.
